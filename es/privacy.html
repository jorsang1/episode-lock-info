Política de Privacidad

Última actualización: 4 de abril de 2026

Resumen

EpisodeLock es una extensión para navegadores que ayuda a los padres a limitar los episodios diarios de streaming en Netflix, Disney+, Amazon Prime Video, YouTube y YouTube Kids. Esta política de privacidad explica cómo la extensión gestiona los datos.

Recopilación de Datos

EpisodeLock no recopila ningún dato personal.

  • No se crean cuentas de usuario.
  • No se envían datos a ningún servidor.
  • No se recopilan análisis ni telemetría.
  • No se utilizan servicios de terceros.

Almacenamiento de Datos

Todos los datos se almacenan únicamente de forma local en tu dispositivo, utilizando la API integrada del navegador chrome.storage.local. Estos datos nunca abandonan tu navegador.

La siguiente información se almacena localmente:

Dato Propósito
Límite de episodios (un número) Tu límite diario de episodios configurado
Contador de episodios (un número) Cuántos episodios se han visto hoy
PIN (cifrado) PIN de control parental para desbloquear o cambiar la configuración
Fecha del último reinicio Para determinar cuándo reiniciar el contador diario

Estos datos solo son accesibles desde la propia extensión y nunca se transmiten a ningún lugar.

Explicación de los Permisos

storage

Se utiliza para guardar tu configuración (límite de episodios, contador diario, PIN) localmente en tu dispositivo. No se utiliza almacenamiento remoto.

Permisos de host (netflix.com, disneyplus.com, amazon.com, primevideo.com, youtube.com, youtubekids.com)

Necesario para detectar cuándo comienza a reproducirse un vídeo en estas plataformas de streaming, para que la extensión pueda contar los episodios y mostrar la superposición del PIN cuando se alcanza el límite diario. La extensión no lee, registra ni transmite ningún contenido de estas páginas.

Privacidad de los Menores

EpisodeLock es una herramienta de control parental diseñada para ser configurada por padres o tutores. No recopila datos de menores ni de ninguna otra persona.

Cambios en Esta Política

Si esta política cambia en una versión futura, la fecha de «Última actualización» indicada arriba se actualizará y los cambios se describirán en las notas de la versión de la extensión.
